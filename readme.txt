Project: My-Theatre-Seating-System
Language: Java
Version: 1.8 update 121

Instructions to run the application
------------------------------------
Please follow below steps to run the application

1. Extract the my-theatre-seating-system.zip file into any folder
2. Give the full path of the input.txt file present in /my-theatre-seating-system
   for Ex: E:\my-theatre-seating-system\input.txt
3. If Possible, import the project in eclipse 
4. Pass the filename as input via "program arguments" and Run TheatreSeatingSystem.java
5. Output will be generated in output.txt file in the path /my-theatre-seating-system
